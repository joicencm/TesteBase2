<h1>Teste Automação Web</h1>

## Sobre

Esse projeto foi desenvolvido, participando do processo seletivo Base2, com intuito de mostrar meus conhecimento.
<br>Teste Automação Web

## Tecnologias Utilizadas

Linguagem Java, utilizando IDE Eclipse

Formato Page Objects

Frameworks:

- Selenium (utlização da classe WebDriver - fazendo aplicações Web)
- Cucumber (utilizando a estrutura Gherkins - BDD)
- JUnit (para a execução do teste)

## Execução dos teste

#####Realizado dois cesarios:

**Login**

1º @login1 - Login com informação valida

2º @login2 - login com informação incorreta

3º @login - Executa ds dois (@login1 + @login2)

**Criar Tarefa**

1º @criarTarefa1 - Campos preenchidos


## Resultado dos teste/bugs

**Login**

- @login1 - Realizado com sucesso
- @login2 - Mensagem esperada

**Criar Tarefa**

- @criarTarefa1 - Operacao realizada com sucesso

#### **Obs**
- A pagina informando Operacao realizada com sucesso (fecha muito rapido). Não sendo possivel "AINDA", a validação da mensagem.
